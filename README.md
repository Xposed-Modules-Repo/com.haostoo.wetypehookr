 # WeTypeHookR

> 📳 **LSPosed 模块** | 为微信输入法提供更丰富的震动选项。

我很喜欢vivo输入法那个双段震动，但是用词习惯和别的小细节让我想用微信输入法。遂制作此模块给微信输入法也弄一个两段震动。

需要API 101。

## 🏖️食用方法

- 在管理器中启用模块后重启输入法。通过**微信输入法设置->帮助与反馈**打开寄生设置页，调整设置，**保存**并**重启输入法** 。

- 使用模块震动时可以关闭输入法自带的按键反馈震动，在测试设备上同时开启自带震动和模块按下震动时，自带震动会被消费掉而无效。

- 模块的配置文件位于``/storage/emulated/0/Android/data/com.tencent.wetype/files/haostoo/``

- 兼容美化模块和部分修改版。
