# WeTypeHookR

> 📱 **LSPosed 模块** | 微信输入法震动增强。

我很喜欢vivo输入法那个双段震动，但是用词习惯和别的小细节让我想用微信输入法。遂制作此模块给微信输入法也弄一个两段震动。

需要API 101。

## 🏖️食用方法

在管理器中启用模块后重启输入法。第一次弹出键盘界面按键会显示toast即hook成功，此时按键理论上就有两段震动了。

可以在模块应用中进行简单的配置。重启输入法生效。
